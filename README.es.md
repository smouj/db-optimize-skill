# Database Optimizer

[![EN](https://img.shields.io/badge/EN-English-blue)](README.md)

## Qué Resuelve

Optimiza consultas de PostgreSQL y MySQL, crea índices y mejora el rendimiento de base de datos.

## Cuándo Se Activa

- Consultas lentas, problemas de rendimiento
- Tuning de PostgreSQL, optimización de MySQL
- Creación de índices, análisis de queries
- Configuración de connection pool

## Requisitos

- Credenciales de acceso a la base de datos
- Réplica de solo lectura para pruebas (recomendado)

## Ejemplos

```
Optimizar consulta SELECT lenta
Crear índice para cláusula WHERE
Analizar resultado de EXPLAIN
Ajustar configuración de PostgreSQL
```

## Limitaciones

- Requiere credenciales de base de datos
- Algunos cambios necesitan downtime
